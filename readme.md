# 张七七的游戏乐园 (Web Games Collection)

一个纯前端的 HTML5 益智与策略游戏合集，支持桌面和移动端。

> **仓库地址**：https://github.com/zhiang-xu/webpagegames2.0

## 技术架构

### 共享组件

| 文件 | 说明 |
|------|------|
| `shared/game-ui.js` | 浮动工具栏（返回主页、音效开关）、Web Audio 音效引擎 |
| `shared/boardgame-ai.js` | 棋类 AI 通用工具：置换表、走法排序、局面评估 |

### 技术亮点

- **纯前端**：无需后端，单 HTML 文件即可运行
- **响应式设计**：适配桌面端和移动端（触屏滑动支持）
- **自定义 CSS**：无外部 UI 框架依赖
- **Web Audio API**：程序化生成游戏音效
- **Canvas API**：贪吃蛇、俄罗斯方块、迷宫等游戏渲染
- **localStorage**：分数和设置持久化

### UI 统一规范

所有游戏页面均集成 `shared/game-ui.js`，提供统一的：

- **浮动工具栏**：右上角返回主页按钮、音效开关按钮
- **音效系统**：tap、select、move、drop、clear、success、win、error、lose、pause 等音效
