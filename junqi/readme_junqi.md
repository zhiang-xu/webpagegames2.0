# 军棋 / 陆战棋 (Junqi)

## 功能介绍

经典军棋（陆战棋）对抗游戏。支持**明棋**和**暗棋**两种模式。

### 核心功能

- **模式切换**：明棋（可见所有棋子）/ 暗棋（敌方棋子显示为背面）
- **胜负判定**：占领对方军旗即获胜
- **战斗日志**：实时记录每步操作和战斗结果
- **音效反馈**：使用共享 `game-ui.js` 音效引擎

### 棋盘布局

- **10 行 × 12 列**标准军棋棋盘
- **地形区域**：
  - 铁路（第 0、1、8、9 行）：工兵可飞，其他棋子不可用
  - 河流（第 3、6 行中间两列）：工兵可过，其他棋子不可进入
  - 公路（其余格子）：标准移动区域
- **大本营**（8 个）：敌方棋子进入即被吃
- **陷阱**（4 个）：进入后降低防御（可被任意己方棋子吃）
- **军旗大本营**（4 个）：仅工兵可进入占领

### 棋子配置

每方 **25 颗棋子**：

| 等级 | 棋子 | 数量 | 说明 |
|------|------|------|------|
| 10 | 司令 | 1 | 最高级 |
| 9 | 军长 | 1 | - |
| 8 | 师长 | 1 | - |
| 7 | 旅长 | 1 | - |
| 6 | 团长 | 1 | - |
| 5 | 营长 | 1 | - |
| 4 | 连长 | 1 | - |
| 3 | 排长 | 1 | - |
| 2 | 工兵 | 3 | 可飞铁路、排雷、占旗 |
| 1 | 地雷 | 4 | 不能主动移动，被炸同归于尽 |
| 1 | 炸弹 | 2 | 遇任何棋子同归于尽 |
| 1 | 军旗 | 1 | 被占即负 |

### 战斗规则

- **炸弹**：遇任何棋子同归于尽
- **地雷**：被工兵排掉，被炸弹炸掉，被炸弹排掉，其他棋子撞上即死
- **等级吃子**：高等级吃低等级，同级同归于尽
- **陷阱**：进入对方陷阱区后，被任意己方棋子吃

### 移动规则

| 棋子 | 移动方式 |
|------|----------|
| 工兵 | 可在铁路直线飞行，可入河、排雷、占旗 |
| 军旗/地雷/炸弹 | 不能移动 |
| 其他棋子 | 公路一步（上下左右） |
