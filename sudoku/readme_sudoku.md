# 数独 (Sudoku)

## 功能介绍

经典数独游戏，支持 3 档难度，集成提示和检查功能。

### 核心功能

- **难度选择**：
  - 简单：挖空 30 格
  - 中等：挖空 40 格
  - 困难：挖空 50 格
- **实时计时**：自动计时
- **错误限制**：3 次错误即游戏结束
- **提示功能**：3 次提示，可自动填入一个空格
- **检查功能**：高亮当前所有错误
- **放弃功能**：显示完整答案
- **高亮机制**：
  - 选中格子高亮
  - 同行/同列/同宫格子浅色高亮
  - 相同数字高亮
- **键盘支持**：1-9 数字键、方向键移动
- **音效反馈**：使用共享 `game-ui.js` 音效引擎

### 游戏流程

1. 选择难度，生成新谜题
2. 点击格子选中
3. 点击数字键盘填入
4. 点击检查高亮错误
5. 3 次错误自动结束
6. 填满且无错即胜利

### 视觉效果

- 粗线分隔 3×3 宫格
- 固定数字灰色背景
- 用户填入数字蓝色
- 错误数字红色高亮
- 胜利弹窗显示用时、错误次数和星级
