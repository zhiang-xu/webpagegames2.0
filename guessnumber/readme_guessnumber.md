# 猜数字 (Guess the Number)

## 功能介绍

经典猜数字游戏。支持键盘和触屏输入。

### 核心功能

- **难度选择**：
  - 简单：范围 1-50
  - 中等：范围 1-100
  - 困难：范围 1-200
- **提示反馈**："太大了" 或 "太小了"
- **历史记录**：显示所有已猜数字
- **最佳成绩**：localStorage 记录每个难度的最少次数
- **评分系统**（基于最优二分法次数）：
  - 3 星：次数 ≤ log₂(max) + 2
  - 2 星：次数 ≤ log₂(max) + 5
  - 1 星：其余情况
- **触屏键盘**：0-9 数字按钮、清除、提交
- **音效反馈**：使用共享 `game-ui.js` 音效引擎

### 游戏流程

1. 选择难度，系统随机选定目标数字
2. 输入猜测数字
3. 系统提示"太大了"或"太小了"
4. 根据提示缩小范围继续猜
5. 猜中即胜利
