# 2048 (2048)

## 功能介绍

经典 2048 数字合并游戏。支持键盘和触屏操作。

### 核心功能

- **操作方式**：方向键 / WASD / 触屏滑动
- **分数系统**：每次合并累加到总分
- **最高分记录**：localStorage 持久化存储
- **胜利机制**：合成 2048 触发胜利弹窗，可继续挑战
- **继续挑战**：达成 2048 后可选择继续玩至无法移动
- **游戏结束判定**：
  - 棋盘填满
  - 且无相邻相同数字
- **移动端按钮**：触屏时显示方向按钮

### 计分规则

合并时，获得合并后数字的分数（如 4+4=8，获得 8 分）。

### 视觉效果

- 经典 2048 配色方案
- 数字越大颜色越深
- 新方块滑入动画（`pop` 动画）
- 合并脉冲动画
- 2-2048 不同颜色区分
