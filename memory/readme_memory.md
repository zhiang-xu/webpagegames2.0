# 记忆卡片 (Memory)

## 功能介绍

经典记忆配对游戏。支持多种难度和卡片主题。

### 核心功能

- **难度选择**：
  - 简单：4×3（6 对卡片）
  - 中等：4×4（8 对卡片）
  - 困难：5×4（10 对卡片）
  - 地狱：6×4（12 对卡片）
- **卡片主题**：emoji 表情符号
- **计时系统**：游戏开始自动计时
- **步数统计**：记录翻开卡片的次数
- **评分系统**：
  - 3 星：步数 ≤ 卡片对数 × 2
  - 2 星：步数 ≤ 卡片对数 × 3
  - 1 星：其余情况
- **胜利弹窗**：显示用时、步数和星级
- **音效反馈**：使用共享 `game-ui.js` 音效引擎

### 游戏流程

1. 选择难度，所有卡片背面朝上
2. 翻开第一张卡片
3. 翻开第二张卡片
4. 匹配成功 → 卡片保持翻开，高亮动画
5. 匹配失败 → 短暂停顿后自动翻回
6. 配对全部完成即胜利

### 视觉效果

- 3D 翻转动画（CSS transform）
- 匹配成功：绿色光效
- 匹配失败：红色闪烁
- 卡片背面使用彩色图案
