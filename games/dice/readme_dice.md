# 骰子游戏 (Dice Games)

## 功能介绍

一站式骰子游戏合集，包含四种经典玩法，深色主题界面，骰子摇动动画。

### 四种模式

**猜大小**
- 三颗骰子，总和 ≥ 11 为大，≤ 10 为小
- 选大或选小后掷骰子，猜对 +1 分，猜错 -1 分

**比大小**
- 玩家 vs 电脑各掷三颗骰子比点数总和
- 赢 +2 分，输 -1 分，平局不变

**猜单双**
- 三颗骰子，总和单数为单，双数为双
- 猜对 +1 分，猜错 -1 分

**21点骰子版**
- 点「开始游戏」各发一颗骰子
- 可随时「再来一颗」补骰，或「够了，停」结束
- 玩家停后电脑自动补牌（< 16 必补，≥ 16 停）
- 最接近 21 点获胜，超过 21 爆掉直接输
- 赢 +2 分，输 -1 分

### 视觉效果

- 深蓝夜空渐变背景（#1a1a2e → #16213e → #0f3460）
- 纯 CSS 骰子：3×3 网格圆点，磨砂白底 + 阴影
- 骰子弹跳动画：物理引擎驱动的高速旋转 + 随机震动 + 弹跳落定，空白面翻滚，结束时揭示点数
- 胜负文字带光晕特效（青色=赢，红色=输，金色=平局）
