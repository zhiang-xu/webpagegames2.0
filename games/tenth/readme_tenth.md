# 十点半 (Ten and a Half)

## 游戏规则

- 一副标准扑克牌（去掉大小王），共52张
- 玩家和电脑各发1张明牌
- 点数计算：2-10按面值，J/Q/K=0.5，A=1或0.5（自动取最优，但不能超过10.5）
- 玩家可随时「要牌」补牌，或「停牌」结束
- 超过10.5立刻爆掉，直接输掉本局
- 玩家停牌后，电脑决定是否要牌：
  - 点数 < 7：必补
  - 点数 7 ~ 8.5：随机决定
  - 点数 >= 9：必停
- 最接近10.5获胜，双方都没爆则比点数
