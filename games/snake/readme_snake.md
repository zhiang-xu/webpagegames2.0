# 贪吃蛇 (Snake)

## 功能介绍

经典贪吃蛇游戏。支持键盘和触屏操作。

### 核心功能

- **难度选择**：
  - 简单：速度 150ms
  - 中等：速度 100ms
  - 困难：速度 70ms
  - 地狱：速度 50ms
- **分数系统**：吃一个食物 +10 分（×难度倍数）
- **最高分记录**：localStorage 持久化存储
- **音效反馈**：吃食物、移动、撞墙、死亡音效
- **新游戏**：随时重新开始
- **暂停/继续**：空格键或点击暂停按钮

### 操作方式

- **键盘**：方向键 / WASD
- **触屏**：屏幕方向按钮（移动端显示）

### 游戏规则

- 蛇每移动一格检测碰撞
- 吃食物身体增长一节
- 撞墙或撞自己即游戏结束
- 食物随机生成在新位置（不与蛇体重叠）

### 视觉效果

- Canvas 渲染
- 蛇身颜色渐变（头部深绿色 → 尾部浅绿色）
- 食物红色闪烁效果
- 死亡时蛇身变为灰色
