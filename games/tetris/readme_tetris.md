# 俄罗斯方块 (Tetris)

## 功能介绍

经典俄罗斯方块游戏。支持键盘和触屏操作。

### 核心功能

- **难度选择**：1-10 级，影响下落速度
- **方块预览**：显示下一个即将落下的方块
- **方块旋转**：顺时针旋转，支持墙踢（Wall Kick）
- **Hard Drop**：一键落到底部
- **暂停/继续**：空格键或暂停按钮
- **新游戏**：随时重新开始
- **音效反馈**：使用共享 `game-ui.js` 音效引擎

### 7 种方块（Tetroimino）

| 形状 | 名称 | 颜色 |
|------|------|------|
| I | I形 | 青色 |
| O | O形 | 黄色 |
| T | T形 | 紫色 |
| S | S形 | 绿色 |
| Z | Z形 | 红色 |
| J | J形 | 蓝色 |
| L | L形 | 橙色 |

### 计分规则

| 操作 | 分数 |
|------|------|
| 消除 1 行 | 100 × 难度 |
| 消除 2 行 | 300 × 难度 |
| 消除 3 行 | 500 × 难度 |
| 消除 4 行（Tetris） | 800 × 难度 |
| Hard Drop | 2 × 落下的行数 |

### 墙踢规则

当旋转后与墙壁或已落方块重叠时，尝试偏移（-1, +1, -2, +2）重新检测，寻找可放置的位置。

### 视觉效果

- Canvas 渲染
- 不同颜色区分方块类型
- 已填满的行闪烁后消除
- 网格线辅助对齐
