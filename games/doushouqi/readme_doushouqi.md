# 斗兽棋 (Dou Shou Qi)

## 功能介绍

经典斗兽棋（动物棋）对抗游戏。7×9 棋盘，8 种动物按等级搏斗。

### 核心功能

- **游戏模式**：人机对弈 / 双人对弈
- **难度选择**：入门 / 初级 / 中级 / 高级（搜索深度 2-5）
- **音效反馈**：集成自定义 SoundManager 和共享 `game-ui.js`

### 棋盘布局

- **7 行 × 9 列**特殊棋盘
- **地形区域**：
  - 河流（4 格）：仅鼠可入
  - 桥（2 格）：所有动物可通行
  - 兽穴/兽洞（2 个）：己方动物可入，非己方进入则吃
  - 陷阱（4 个）：进入后降低防御等级

### 棋子配置

每方 8 颗动物：

| 等级 | 动物 | 说明 |
|------|------|------|
| 8 | 狮 | 最高级，可跳河吃对面 |
| 7 | 虎 | 可跳河吃对面 |
| 6 | 豹 | - |
| 5 | 狼 | - |
| 4 | 狗 | - |
| 3 | 猫 | - |
| 2 | 鼠 | 最低级，可入河，可吃狮虎 |
| 1 | 大象 | 最高防御，不能入河 |

### 吃子规则

- 高等级吃低等级
- 同等级同归于尽
- **特殊规则**：
  - 鼠入河后，不能吃河外动物（除非在桥上）
  - 鼠入河后可吃狮、虎
  - 狮、虎可跳跃吃河对岸的鼠、狮、虎（同等级直线无阻挡）
  - 大象不能入河，但可吃河外鼠

### 胜负判定

- 占领对方兽穴即获胜
- 己方所有动物被吃光则负
