# Flappy Bird（像素鸟）

## 功能介绍

经典 Flappy Bird 玩法。点击或按键让小鸟往上飞，穿过绿色水管间隙得分，撞上水管或触地则 Game Over。

### 核心功能

- **四种难度**：
  - 简单：水管速度慢，间隙宽（160px）
  - 普通：速度中等，间隙适中（130px）
  - 困难：速度快，间隙较窄（105px）
  - 疯狂：水管速度快，间隙极窄（82px）
- **最高分记录**：各难度独立 localStorage 存储
- **音效反馈**：使用共享 `game-ui.js` 音效引擎（点击音、得分音、失败音）
- **键盘支持**：空格键 / 上方向键
- **触屏支持**：点击画布任意位置

### 视觉效果

- 蓝色渐变天空背景
- 漂浮白云动画（不同大小和透明度）
- 远处绿色小山剪影
- 渐变绿色水管，带高光细节
- 像素风黄色小鸟：翅膀扇动动画、上下飞行旋转、橙红嘴巴、头顶呆毛
- 地面带纹理竖条纹，持续滚动
- 画面中央悬浮大字计分
